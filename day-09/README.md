# DAY 09: Silent Auction

## 📌 Description
A Python program that simulates a silent auction system.

Users can enter their names and bid amounts, and the program stores all bids in a dictionary. At the end of the auction, the program determines and displays the highest bidder.

This project focused heavily on working with dictionaries, loops, functions, and conditional logic.

---

## 🚀 Concepts Practiced

- Dictionaries
- Key-value pairs
- While loops
- Functions
- Conditional statements
- Iterating through dictionaries
- Finding the highest value in a dataset
- Input validation

---

## 🧠 What I Learned

In this project, I learned how to:

- Store dynamic user data inside dictionaries
- Loop through dictionary entries
- Track the highest value while iterating through data
- Organize code using functions
- Improve program flow and user experience
- Handle invalid user input

---

## ▶️ How It Works

1. The program asks the user for:
   - their name
   - their bid amount

2. The bid is stored in a dictionary.

3. The program asks if there are more bidders.

4. Once the auction ends, the program determines the highest bidder and displays the winner.
