# 🐍 100 Days of Python Challenge

This repository contains my progress through the **100 Days of Code: Python Bootcamp** by Dr. Angela Yu on Udemy.

The goal of this challenge is to build **100 Python projects in 100 days**, covering everything from beginner fundamentals to more advanced topics like automation, web development, and data science.

---

## 🚀 About the Challenge

- 📚 Course: 100 Days of Code - Python Bootcamp
- 👩‍🏫 Instructor: Dr. Angela Yu
- 🎯 Goal: Improve Python skills by building real projects daily
- ⏳ Duration: 100 Days
- 🧠 Topics covered:
  - Python fundamentals
  - Object-Oriented Programming (OOP)
  - APIs & Web Scraping
  - Automation
  - Data Analysis
  - Web Development (Flask)
  - Game Development
