# 🎯 Number Guessing Game

Day 12 project from the **100 Days of Code: Python Bootcamp** course.

A simple command-line game where the computer randomly selects a number between 1 and 100, and the player must guess it before running out of attempts.

## 🚀 Features

* Random number generation
* Easy and Hard difficulty modes
* Feedback after each guess

  * Too high
  * Too low
  * Correct answer
* Attempt tracking
* Win and lose conditions
* Practice with functions, loops, conditionals, and scope

## 🛠️ Technologies Used

* Python 3
* `random` module

## 📚 Concepts Practiced

* Functions
* Return values
* Local and global scope
* Conditional statements
* While loops
* User input
* Random number generation

## 🎮 How to Play

1. Run the program.
2. Choose a difficulty:

   * Easy → 10 attempts
   * Hard → 5 attempts
3. Enter your guesses.
4. Use the feedback to narrow down the correct number.
5. Guess the number before your attempts run out!

## Example Gameplay

```text
Choose a difficulty: Type 'easy' or 'hard': hard

You have 5 attempts remaining.
Guess a number: 50
Too low.

You have 4 attempts remaining.
Guess a number: 75
Too high.

You have 3 attempts remaining.
Guess a number: 63
You got it!
```

## 🎯 What I Learned

This project helped me understand the difference between printing values and returning values from functions. It also reinforced how local and global scope work in Python and how functions can be used to organize game logic into smaller, reusable pieces.
