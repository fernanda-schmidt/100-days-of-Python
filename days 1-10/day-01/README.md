# 🎸 Day 1: Band Name Generator

## 📌 Description
A simple Python program that generates a band name based on user input.

The program asks for the user's hometown and childhood pet, then combines them to create a fun band name.

## 🧠 What I Learned
- Using `input()` to capture user data
- String formatting with f-strings
- Improving output formatting with `.title()`
