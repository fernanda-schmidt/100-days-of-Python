# 🎯 Day 10: Simple Calculator

## 📌 Description
A simple command-line calculator built in Python as part of the 100 Days of Python challenge.

The program allows users to perform basic mathematical operations and continue calculations using previous results.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Continuous calculations
- Input validation for operations
- Restart or quit options

## 🧠 What I Learned
- Creating and using functions
- Storing functions inside dictionaries
- Using loops to control program flow
- Handling user input
- Basic error handling and validation

## ▶️ How to Run

```bash
python main.py
```
